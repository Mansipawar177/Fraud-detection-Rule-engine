# Fraud Detection Rule Engine

## Project Description
The Fraud Detection Rule Engine is a Spring Boot application that detects suspicious and fraudulent financial transactions based on predefined fraud rules.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Lombok
- Swagger / OpenAPI
- Postman

## Features
- Transaction CRUD APIs
- Fraud Rule CRUD APIs
- Fraud Result APIs
- Fraud Risk Evaluation
- Duplicate Transaction Detection
- Global Exception Handling
- RESTful APIs
- Database Integration

## API Endpoints
### Transactions
- POST /transactions
- GET /transactions
- GET /transactions/{id}
- PUT /transactions/{id}
- DELETE /transactions/{id}

### Fraud Rules
- POST /fraud-rules
- GET /fraud-rules
- GET /fraud-rules/{id}
- PUT /fraud-rules/{id}
- DELETE /fraud-rules/{id}

### Fraud Results
- GET /fraud-results
- GET /fraud-results/{id}

## Database
- MySQL
